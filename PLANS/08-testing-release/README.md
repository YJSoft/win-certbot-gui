# 8단계: 테스트/검증/릴리즈 계획

## 테스트 전략
- 단위 테스트: ACME 래퍼, challenge 핸들러, 유틸
- 통합 테스트: 스테이징 ACME end-to-end
- E2E 테스트: GUI 마법사 흐름(Playwright 등)

## 주요 검증 시나리오
1. 단일 도메인 HTTP-01 발급 성공
2. 단일 도메인 DNS-01 발급 성공
3. wildcard DNS-01 발급 성공
4. wildcard에서 HTTP-01 선택 차단 확인
5. DNS 전파 지연/실패 처리
6. 갱신 자동 실행 및 결과 반영

## 운영/릴리즈
- 베타 채널 운영(초기 사용자 피드백)
- 크래시/오류 리포팅 연동
- 버전별 마이그레이션 정책

## 문서화
- 사용자 가이드(HTTP/DNS 각각)
- 운영자 가이드(자동화/보안)
- 문제 해결(FAQ)

## 산출물
- 테스트 리포트 템플릿
- 릴리즈 노트 템플릿
- 배포 승인 체크리스트
