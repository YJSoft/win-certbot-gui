# 6단계: UI/UX 상세 설계

## 핵심 화면
1. 대시보드
   - 발급된 인증서 목록
   - 만료 임박 경고
2. 새 인증서 발급 마법사
   - 도메인 입력
   - Challenge 방식 선택
   - 검증 진행
   - 발급 완료
3. 설정
   - Let's Encrypt 환경(스테이징/프로덕션)
   - 기본 저장 경로
4. 로그/진단
   - 최근 작업 기록
   - 오류 상세 보기

## 마법사 규칙
- 도메인 입력 시 wildcard 자동 감지
- wildcard 포함 시 HTTP 옵션 비활성화 + DNS 필요 안내
- 도메인별 개별 상태 표시

## 사용자 보호 장치
- 스테이징 환경 기본값으로 시작(초기 실수 방지)
- 프로덕션 전환 시 확인 팝업
- 위험작업(키 삭제 등) 이중 확인

## 산출물
- 와이어프레임
- 상태/에러 메시지 카피 초안
- 접근성 체크리스트
