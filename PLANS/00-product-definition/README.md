# 0단계: 제품 정의 (Product Definition)

## 프로그램 기본 정보
- **코드네임**: `win-certbot-gui`
- **프로그램명(가칭)**: **WinCertbot Manager**
- **대상 OS**: Windows 10/11 (x64 우선, 이후 ARM64 검토)
- **핵심 목적**: Electron 기반 GUI로 Let's Encrypt ACME 서버와 연동해 인증서 발급/갱신 자동화를 지원

## 핵심 요구사항
1. ACME 프로토콜로 Let's Encrypt와 통신
2. 도메인 인증(Challenge) 방식 지원
   - `HTTP-01`: 지정 파일을 웹서버에 업로드해 검증
   - `DNS-01`: DNS TXT 레코드 추가로 검증
3. **와일드카드 도메인(`*.example.com`)은 DNS-01만 허용**
4. 인증서 발급/갱신 결과를 GUI에서 확인 가능
5. 초보자도 따라할 수 있는 단계형 마법사 제공

## 기능 범위 (MVP)
- 새 인증서 발급
- 기존 인증서 갱신
- 인증서/키 파일 내보내기(PEM)
- 계정 키 생성 및 저장
- Challenge 수행 가이드/자동화(가능 범위 내)
- 작업 로그/오류 로그 표시

## 비기능 요구사항
- 보안: 개인키 안전 저장(Windows DPAPI 또는 OS 안전 저장소 사용)
- 신뢰성: 실패 시 재시도/롤백/명확한 오류 메시지
- 사용성: 도메인 입력/검증 방식 선택을 직관적으로 제공
- 확장성: DNS 공급자 플러그인 구조
